上传指南（推荐使用Vercel，微信兼容性最好）：

  1. 访问 Vercel 并登录（可使用GitHub账号）
  2. 点击 "New Project" → "Upload"
  3. 拖拽 index.html 文件到上传区域，或点击 "Select" 选择文件
  4. 点击 "Deploy" 开始部署（约1-2分钟）
  5. 部署完成后，获取生成的链接（如 https://share-link.vercel.app）
  6. 在微信中发送该链接，会自动显示分享卡片
  7. 用户点击卡片后会自动跳转到原文章链接

其他上传选项：

  • GitHub Pages：创建公开仓库，上传文件并开启Pages功能
  • Netlify：拖拽文件上传，获取分享链接
  • Coding Pages：国内访问速度快，适合微信分享

注意事项：

  • 图片URL必须是HTTPS且可公开访问的链接
  • 推荐使用图床（如 Imgur）上传图片获取稳定链接
  • 首次分享到微信可能需要等待5-10分钟让微信爬虫抓取
  • 如果分享后不显示卡片，尝试清除微信缓存后重试
